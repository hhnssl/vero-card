### 숙련도가 떨어지는 개념을 복습하라고 한다

현재 시간을 기준으로 어떤 개념이... 다시 복습해야 하는 개념, 다시 복습해야 하는 자료가 된다

<-> 복습하지 않아도 되는 개념. 완전히 숙달된 개념

숙련도란 사람마다 다르다.
숙련도란 개념이나 자료 하나에 대한 것이다
숙련도란 점수이며, 문제를 하나 풀면 1단계 올라간다다

0, 1, 2, 3, 4, 5

- 맞고 맞고 틀리고 틀리고 0
- 맞고 맞고 틀리고 맞고 2
- 틀리고 틀리고 맞고 맞고 2

초기 상태 = 0점
- 맞으면 -1
- 틀리면 +1
- "0이하로 떨어질 수 없다"!!!
-> 시간 순서가 존재함
